# Code Debug Skill

当用户提供报错信息、traceback、日志、异常行为描述时使用本技能。

## 使用步骤
1. 先读取用户输入中的报错、日志、代码片段
2. 判断错误类型：
   - 语法错误
   - 导入错误
   - 环境错误
   - 网络/API 错误
   - 工具调用格式错误
3. 如需验证，调用 Python 执行工具做最小复现
4. 给出：
   - 问题原因
   - 修改建议
   - 修改后的代码
   - 验证方式

## 约束
- 不要编造不存在的报错上下文
- 需要说明“确定项”和“推测项”
- 优先给最小修改方案

## 输出格式
1. 问题定位
2. 原因分析
3. 修改方案
4. 验证步骤
