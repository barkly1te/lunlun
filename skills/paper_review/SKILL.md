# Paper Review Skill

当用户要求审稿、润色、分析论文段落、判断是否符合顶刊/顶会写作风格时使用本技能。如果用户要求你帮他润色论文，你也应该使用这个skill。

## 使用步骤
1. 读取论文片段或文档
2. 提取：
   - 研究问题
   - 方法主线
   - 贡献表达
   - 实验叙事
3. 从以下维度评价：
   - 结构清晰度
   - 学术表达密度
   - 术语统一性
   - 图文一致性
   - venue 风格适配
4. 输出：
   - 总评
   - 问题清单
   - 改写建议
   - 示例改写

## 约束
- 不要脱离用户提供的文本瞎补内容
- 结论必须可追溯到原文
